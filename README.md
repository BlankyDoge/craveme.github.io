# Privacy and Policy CraveMe
CraveMe menghargai privasi pengguna dan berkomitmen melindungi data pribadi sesuai Undang-Undang Perlindungan Data Pribadi Indonesia.

# Data yang Dikumpulkan
Aplikasi mengumpulkan informasi seperti email, preferensi resep, dan data penggunaan (IP address, jenis perangkat) untuk personalisasi resep dan peningkatan layanan. Data ini termasuk informasi yang Anda berikan saat registrasi atau pencarian resep, serta data otomatis dari interaksi aplikasi. Kami tidak mengumpulkan data sensitif seperti informasi kesehatan.

# Penggunaan Data
Data digunakan untuk menyediakan resep, mengirim notifikasi, menganalisis tren penggunaan, dan meningkatkan aplikasi. Informasi juga mendukung pemrosesan transaksi jika ada fitur pembelian bahan resep di masa depan. Kami tidak menjual data kepada pihak ketiga tanpa persetujuan.​​

# Berbagi Data
Data hanya dibagikan dengan penyedia layanan terpercaya (seperti hosting cloud) untuk operasional aplikasi, atau jika diwajibkan hukum. Pengguna dapat memilih berbagi resep favorit secara publik melalui fitur yang ada. Tidak ada pengungkapan ke pihak ketiga untuk pemasaran tanpa izin eksplisit.​
